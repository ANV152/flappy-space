In dit project laat ik zien samen met mijn klasgenoot Tjees, hoe sommige design patronen kunnen worden toegepast waaronder factory en command patterns. Het doel van de game is de hoogste score halen door zo veel mogelijk asteroids te vernietigen, maar je moet opletten dat Flappy niet terug op de Aarde valt. Hij gebruikt namelijk een spacesuit en een jet propulsion met weinig brandstof, dus wees zuinig ermee en ontwijk alle asteroids om te overleven
